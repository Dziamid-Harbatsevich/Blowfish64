# Руководство пользователя к приложению ШПС-Б64.

Приложение ШПС-Б64 разработано 07.09.2023 г., и реализует шифрование текста с помощью алгоритма Blowfish.

Функционал приложения:
	- кодирование/декодирование текста, в т.ч. из файла;
	- загрузка/сохранение текста из файла/в файл как зашифрованного, так и исходного;
	- вывод данных о количестве символов текста зашифрованного/исходного;
	- генерирование ключа шифрования;
	- загрузка/сохранение ключа шифрования из файла/в файл;
	- авто-очищение полей ввода данных в исходное состояние.

Горячие клавиши:
Ctrl+X - Вырезать текст;
Ctrl+C - Копировать текст;
Ctrl+V - Вставить текст;
Ctrl+X - Отмена действия;
Ctrl+Y - Возврат к отмененному действию;

По умолчанию:
-	ключи сохраняются в папку "keys" с префиксом "key_"
-	исходный текст сохраняется в папку "plain" с префиксом "plain_"
-	шифрованный текст сохраняется в папку "encrypted" с префиксом "encrypted_"
